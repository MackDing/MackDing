### Featured Projects

### 🌐 [opc.ren](https://opc.ren)

📡 **[awesome-ai-api](https://github.com/MackDing/awesome-ai-api)** — Curated list of 200+ AI API providers with daily ranking and live `/v1/models` probing for Claude, DeepSeek, Gemini, and more · [live site](https://mackding.github.io/awesome-ai-api/)

📊 **[opc-daily-signal](https://github.com/MackDing/opc-daily-signal)** — AI-powered daily decision intelligence for OPC founders

📄 **[doc-preprocess-hub](https://github.com/MackDing/doc-preprocess-hub)** — Enterprise document preprocessing pipeline — MinerU + docling + PII redaction + audit console

☸️ **[hermes-manager](https://github.com/MackDing/hermes-manager)** — K8s-native control plane for AI agent fleets, 5-minute Helm install

🔐 **[agent-vault](https://github.com/MackDing/agent-vault)** — HTTP credential proxy and vault for AI agents — zero-secret-exposure tool calls
